# GDPR Analysis

Небольшой анализ текущего статуса (на момент февраля 2025 года) официальных рекомендаций (документов) в Европейском союзе по следующим вопросам:
* получение согласия субъектов на обработку персональных данных;
* передача данных за пределы Европейского союза;
* области применения законодательства по персональным данным (General Data
Protection Regulation, GDPR);
* актуальный перечень стран за пределами ЕС, которые обеспечивают достаточный
уровень защиты персональных данных.
